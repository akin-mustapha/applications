## Project - Overview

### Prompt Manager

A web application for managing AI prompt files and templates.

### Background

The user writes many prompt markdown files and struggles to keep track of them — finding, reusing, and managing templates and history is painful. This app solves that.

### What It Does

- **Prompts** — create, view, update, and delete prompt files
- **Templates** — create and manage reusable prompt templates
